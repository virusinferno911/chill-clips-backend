# 🎬 Chill Clips Backend

**By Oluwasheyi Ojelade | Virusinferno Digital Studio** *Cloud • DevOps • AI Automation*

## 📌 Overview
This repository contains the backend service for the **Chill Clips** movie application. It is a robust REST API built using **Java and Spring Boot**, connected to a secure **MongoDB** database, and configured for deployment on a Linux production server.

## 🚀 Tech Stack
* **Framework:** Java 17 / Spring Boot 3
* **Database:** MongoDB
* **Build Tool:** Maven
* **Deployment:** Nginx & Caddy Reverse Proxy

## 🛠️ Deployment Notes
This application is designed to run securely in the background (using `nohup`) on port 8080, serving JSON data to the React frontend. The database connection is secured and restricted to local network traffic via firewall rules.
